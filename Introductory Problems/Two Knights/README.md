
# Two Knights

**Time limit:** 1.00 s  
**Memory limit:** 512 MB  

---

## Problem

Your task is to count, for each integer \( k = 1, 2, .... , n \), the number of ways to place two knights on a \( k * k \) chessboard so that they do not attack each other.

**Note :**
A knight in chess moves in an "L" shape: two squares in one direction and one square perpendicular to that.

---

### Input :

The only input line contains an integer ( n ).

---

### Output :

- Print \( n \) integers.  

- For each \( k = 1 \) to \( n \), output the number of ways to place two knights on a \( k \times k \) board so that they do not attack each other.

- Each result should be printed on its own line.

---

## Constraints

- \( 1 <= n <= 10000 \)

---

### Example

### Input :
```

8

```

### Output :
```

0
6
28
96
252
550
1056
1848

```
