# [String Matching][1]

* Time limit: 1.00 s
* Memory limit: 512 MB

---

### Statement :

Given a string and a pattern, your task is to count the number of positions where the pattern occurs in the string.

---

### Input :

The first input line contains a string of length `n`.

The second input line contains a pattern of length `m`.

Both strings consist of characters `a-z`.

---

### Output :

Print one integer: the number of occurrences.

---

## Constraints

* `1 <= n, m <= 10^6`

---

## Example

### Input

```text id="j4m8q2"
saippuakauppias
pp
```

### Output

```text id="w7x1n5"
2
```

---

### Explanation

The pattern:

```text id="k9v3r1"
pp
```

appears two times in the string:

```text id="c6m2t8"
saippuakauppias
```
