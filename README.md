# CSES Solutions

Professional, organized solutions for the [CSES Problem Set](https://cses.fi/problemset/) in Java and C++. This repository focuses on clean, well-structured implementations with emphasis on readability and correctness.

## Overview

- **Coverage:** A growing collection of CSES problems solved in **Java** and **C++**.
- **Goal:** Provide reference-quality solutions that are easy to understand and study.
- **Style:** Consistent structure, clear naming, and efficient algorithms.

## Repository Structure

```
.
├── Java
│   └── ... Java solutions ...
└── C++
    └── ... C++ solutions ...
```

> Folder names may vary depending on how solutions are organized by topic or problem set section.

## How to Use

1. Browse the language folder of your choice.
2. Open the solution file for the problem you’re studying.
3. Read the comments and logic flow to understand the approach.
4. Compile and run locally if you want to test or modify.

## Running Solutions

### Java
```bash
javac Solution.java
java Solution < input.txt
```

### C++
```bash
g++ -std=c++17 -O2 -Wall Solution.cpp -o solution
./solution < input.txt
```
